## 快速启动 `krs`
### 创建项目
```javascript
$ npm i mmkrs-cli -g
$ mmkrs -i ---> select project
$ <Your Project Name>
$ cd <Your Project Name>
$ npm i
$ npm run dev //本地开发的watch 模式
$ open http://<Your local ip>:8808
```
### 创建页面

``` javascript
$ cd <Your Project Name>
$ mmkrs -i -----> select page ----> <Your  pageName>
$ open http://<Your local ip>:8808/<Your  pageName>
```
